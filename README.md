## 핵심 기능
1) 회원
- 회원가입
- 회원정보 조회/ 수정
- 로그인 / 로그아웃
- 리뷰 작성
- 예약 내역 확인
2) 검색
- 지역명 / 식당명 / 지도 / 비건유형에 따른 검색
3) 예약
- 예약 신청 / 취소
- 결제
4) 관리자
- 날짜 별 예약자 확인
- 식당 정보 수정
- 취소 / 노쇼 확인
